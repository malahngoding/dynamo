# Javascript

## Fungsi if dan else

Fungsi if dan else adalah suatu fungsi untuk menentukan suatu aksi dalam suatu kondisi. Cara menentukan fungsi if dan else pada php, sebagai berikut:

```javascript
if (kondisi){
	#pengkodean yang digunakan dalam kondisi benar
} else{
	#pengkodean yang digunakan dalam kondisi salah
};
```

## Fungsi else if

Fungsi elseif adalah suatu fungsi untuk menentukan suatu aksi dalam beberapa kondisi. Cara menentukan fungsi elseif pada php, sebagai berikut:

```javascript
if (kondisi 1) {
    #pengkodean yang digunakan dalam kondisi 1 benar
} elseif (kondisi 2) {
    #pengkodean yang digunakan dalam kondisi 1 salah dan kondisi 2 benar
} else {
    #pengkodean yang digunakan dalam kondisi 1 salah dan kondisi 2 benar
}
```

## Fungsi Pushing

Array push berfungsi untuk memasukkan array baru ke dalam array. Penggunaan fungsi array push pada php, sebagai berikut:

```javascript
const namaBuah = [mangga, jeruk, apel, melon, salak]
namaBuah.push('kiwi') // Menambahkan "kiwi" ke namaBuah
```

## Fungsi merging(concatenating) array

Merging (concatenating) berfungsi untuk menggabungkan dua array atau lebih. Penggunaan fungsi array merge pada php, sebagai berikut:

```javascript
const namaPerempuan = ['Sri', 'Ida']
const namaLakiLaki = ['Parjo', 'Milwan', 'Ketut']

// Concatenate (join) myGirls and myBoys
const temanSaya = namaPerempuan.concat(namaLakiLaki)
```

## Fungsi length

Length berfungsi untuk menentukan jumlah array. Penggunaan length pada php, sebagai berikut:

```javascript
const namaBuah = [mangga, jeruk, apel, melon, salak]
namaBuah.length // Returns 5
```

## Loop data memiliki range tertentu

Untuk loop data yang memiliki range tertentu dapat menggunakan fungsi for. Penggunaan fungsi for pada php, sebagai berikut:

```javascript
const namaBuah = [mangga, jeruk, apel, melon, salak]
let namaBuahLength = nama_buah.length

text = '<ul>'
for (let i = 0; i < namaBuahLength; i++) {
  text += '<li>' + namaBuah[i] + '</li>'
}
text += '</ul>'
```

## Loop data berupa array

Untuk loop data berupa array dapat menggunakan fungsi foreach. Penggunaan fungsi foreach pada php, sebagai berikut:

```javascript
const namaBuahh = [mangga, jeruk, apel, melon, salak]
let text = '<ul>'
namaBuah.forEach(myFunction)
text += '</ul>'

function myFunction(value) {
  text += '<li>' + value + '</li>'
}
```

## Membuat database menggunakan MySql dengan Node.js

Sebelum kita dapat mengakses data dari database, kita perlu membuat database terlebih dahulu dengan cara membuat database di MySQL, sebagai berikut:

```javascript
var mysql = require('mysql')

var con = mysql.createConnection({
  host: 'localhost',
  user: 'yourusername',
  password: 'yourpassword',
})

con.connect(function (err) {
  if (err) throw err
  console.log('Connected!')
  con.query('CREATE DATABASE mydb', function (err, result) {
    if (err) throw err
    console.log('Database created')
  })
})
```

## Memasukkan data pada MySQL dengan Node.js

Cara memasukkan data ke dalam tabel profile pada MySql dengan Node.js, sebagai berikut:

```javascript
var mysql = require('mysql')

var con = mysql.createConnection({
  host: 'localhost',
  user: 'yourusername',
  password: 'yourpassword',
  database: 'mydb',
})

con.connect(function (err) {
  if (err) throw err
  console.log('Connected!')
  var sql = "INSERT INTO customers (name, address) VALUES ('Company Inc', 'Highway 37')"
  con.query(sql, function (err, result) {
    if (err) throw err
    console.log('1 record inserted')
  })
})
```

## Mengupdate data pada MySql dengan Node.js

Cara mengupdate data ke dalam tabel profile pada MySql dengan Node.js, sebagai berikut:

```javascript
var mysql = require('mysql')

var con = mysql.createConnection({
  host: 'localhost',
  user: 'yourusername',
  password: 'yourpassword',
  database: 'mydb',
})

con.connect(function (err) {
  if (err) throw err
  var sql = "UPDATE customers SET address = 'Canyon 123' WHERE address = 'Valley 345'"
  con.query(sql, function (err, result) {
    if (err) throw err
    console.log(result.affectedRows + ' record(s) updated')
  })
})
```
