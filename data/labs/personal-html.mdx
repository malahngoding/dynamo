# Personal HTML

Merupakan proyek yang dapat dilakukan ketika mulai belajar HTML.
Dengan menggunakan element HTML dasar seperti `div`, `h1`, `p`, `img`, `li`,`ul`dan `ol`.
Kita dapat membuat profile yang sesuai dengan kepribadian kita.
Selain itu, kita juga dapat melatih skill css kita dengan menulis css tanpa menggunakan bantuan _library_.
