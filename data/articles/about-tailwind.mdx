# Tailwind

Tailwind CSS merupakan framework yang bersifat utilityfirst untuk membangun desain antarmuka khusus dengan cepat dan sesuai yang di inginkan tanpa harus override style seperti di framework CSS lainnya.

Tailwind CSS sangat performance dan berbeda dengan konsep yang lainnya, serta tidak dapat dibandingkan dengan framework CSS lainnya seperti bootsstrap, spectre dll.

### Responsive

Utilitas Tailwind CSS meyiapkan jenis-jenis responsive untuk mempermudah programmer dalam membangun antarmuka responsive tanpa menggunakan CSS khusus.`

### Component-friendly

Tailwind CSS menyediakan alat untuk mengekstraksi kelas komponen dari pola utilitas untuk memudahkan programmer dalam memperbaharui beberapa instances dalam sebuah komponen dari satu tempat.

### Lebih mudah dipahami

Tailwind CSS menyediakan document yang sangat jelas dan mudah dipahami dari mulai cara menginstall hingga penggunaan untuk membuat desain antarmuka.

### Efisien waktu

Programmer adalah tidak perlu lagi untuk memikirkan nama dari class dengan menggunakan Tailwind CSS semua sudah ada di dalam document.

## Mendukung pada browser

Secara keseluruhan Tailwind dirancang dan diuji pada versi stabil terbaru dari Chrome, Firefox, Edge, and Safari. Meskipun sebagian besar fitur di Tailwind CSS akan berfungsi di semua browser modern, Tailwind juga menyertakan API untuk beberapa fitur mutakhir yang belum didukung oleh semua browser, misalnya `:focus-visible` pseudo-class dan `backdrop-filter` utilities.

Karena Tailwind adalah framework tingkat rendah, sangat mudah untuk menghindari fitur-fitur ini jika Anda tidak dapat menggunakannya hanya dengan tidak menggunakan utilitas atau pengubah yang tidak didukung seperti bagaimana Anda tidak akan menggunakan fitur-fitur CSS itu di CSS Anda.

Anda dapat menggunakan database pada sumber yang bagus saat Anda tidak yakin tentang dukungan untuk fitur CSS yang tidak dikenal.
