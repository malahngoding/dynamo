# CSS in JS

CSS kepanjangan dari _Cascading Style Sheet_ adalah bahasa-bahasa yang merepresentasikan halaman website, seperti warna, layout, dan font. Dengan menggunakan CSS, seorang web developer dapat membuat halaman website yang dapat beradaptasi dengan berbagai macam ukuran layar. Sedangkan _JavaScript_ mengimplementasikan fitur yang dirancang untuk mengendalikan bagaimana sebuah halaman web berinteraksi dengan _user_ atau penggunanya.
CSS in JS saat ini semakin umum dalam pengembangan front-end secara modern terutama pada komunitas React.

# 6 Library CSS in JS

Berikut beberapa Library CSS in JSS yang kami ketahui saat ini:

#### 1. Linaria

Dapat menulis CSS di JS dan menggunakan file CSS asli selama pembuatan sehingga memiliki istilah Zero Runtime CSS in JS. Berbasis dynamic prop dengan React bindings dengan cara mengubah ke variabel CSS secara otomatis.

#### 2. Freestyler

Memiliki istilah 5th generation React styling library karena sangat cepat, ramping, dan dengan trilyun fitur.

#### 3. Emotion

Emotion adalah suatu library yang didesain untuk menulis CSS styles dengan javascript. Dapat memberikan pengalaman yang hebat untuk developer telah disediakan fitur source maps, labels dan testing utilities dalam bentuk string. Emotion dapat digunakan pada framework Agnostic.

#### 4. Fela

Fela didesain mampu melakukan universal rendering (client- and server-side rendering) sejak awal, menyediakan fitur Local Namespace yang dapat mengatur nama kelas secara unik di CSS dan menggunakan desain atomic class sehingga memungkinkan ukuran markup terkecil.

#### 5. Styled-components

Styled-components mampu melakukan template yang diberikan tanda (tagged) secara unik dan dapat membuat komponen react secara normal atau biasanya dengan hanya mendefinisikan style-nya.

#### 6. Aphrodite

Aphrodite merupakan suatu framework agnostic CSS-in-JS yang mampu melakukan server-side rendering, browser prefixing, dan minimum CSS generation.
