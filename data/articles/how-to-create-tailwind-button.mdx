## How To Create Tailwind Button

Sebuah `<button>` tag didefinisikan sebagai tombol yang bisa diklik.
Didalam sebuah `<button>` terdapat elemen yang bisa kamu masukan text (dan tags seperti `<i>`, `<b>`, `<strong>`, `<br>`, `<img>`, dll.). Tombol tidak bisa dibuat dengan tag `<input>` element!<br>
Contoh
Penggunaan TailWindCSS untuk memodifikasi tombol:

```javascript
<button className="h-[50px] w-[100px] rounded-full bg-gradient-to-r from-pink-500 via-purple-500 to-blue-500 text-white font-extrabold">
  Create
</button>
```
