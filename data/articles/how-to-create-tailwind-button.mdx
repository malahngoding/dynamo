## How To Create Tailwind Button

The `<button>` tag defines a clickable button.
Inside a `<button>` element you can put text (and tags like `<i>`, `<b>`, `<strong>`, `<br>`, `<img>`, etc.). That is not possible with a button created with the `<input>` element!<br>
Example
Use TailWindCSS to style buttons:

```javascript
<button className="h-[50px] w-[100px] rounded-full bg-gradient-to-r from-pink-500 via-purple-500 to-blue-500 text-white font-extrabold">
  Create
</button>
```
