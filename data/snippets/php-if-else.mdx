## Fungsi if dan else

Fungsi if dan else adalah suatu fungsi untuk menentukan suatu aksi dalam suatu kondisi. Cara menentukan fungsi if dan else pada php, sebagai berikut:

```php
if ($a > $b){
echo "a lebih besar dari b"
} else{
echo "a tidak lebih besar dari b"
};
```
