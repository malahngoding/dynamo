## Fungsi merging(concatenating) array

Merging (concatenating) berfungsi untuk menggabungkan dua array atau lebih. Penggunaan fungsi array merge pada php, sebagai berikut:

```javascript
const namaPerempuan = ['Sri', 'Ida']
const namaLakiLaki = ['Parjo', 'Milwan', 'Ketut']

// Concatenate (join) myGirls and myBoys
const temanSaya = namaPerempuan.concat(namaLakiLaki)
```
