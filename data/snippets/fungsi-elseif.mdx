## Fungsi else if

Fungsi elseif adalah suatu fungsi untuk menentukan suatu aksi dalam beberapa kondisi. Cara menentukan fungsi elseif pada php, sebagai berikut:

```javascript
if (kondisi 1) {
    #pengkodean yang digunakan dalam kondisi 1 benar
} elseif (kondisi 2) {
    #pengkodean yang digunakan dalam kondisi 1 salah dan kondisi 2 benar
} else {
    #pengkodean yang digunakan dalam kondisi 1 salah dan kondisi 2 benar
}
```
