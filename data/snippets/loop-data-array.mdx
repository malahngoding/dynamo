## Loop data berupa array

Untuk loop data berupa array dapat menggunakan fungsi foreach. Penggunaan fungsi foreach pada php, sebagai berikut:

```javascript
const namaBuahh = [mangga, jeruk, apel, melon, salak]
let text = '<ul>'
namaBuah.forEach(myFunction)
text += '</ul>'

function myFunction(value) {
  text += '<li>' + value + '</li>'
}
```
