## Memasukkan data menggunakan PHP MySql

Cara memasukkan data ke dalam tabel profile menggunakan MySql, sebagai berikut:

```php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "ProfileTable";

// Membuat koneksi
$database = new mysqli($servername, $username, $password, $dbname);
// Mengecek koneksi
if ($database->connect_error) {
  die("Gagal terkoneksi: " . $database->connect_error);
}

$sql = "INSERT INTO Profile (firstname, lastname, email)
VALUES ('Galih', 'Indrasari', 'galihindrasari@gmail.com')";

if ($database->query($sql) === TRUE) {
  echo "Data baru berhasil tersimpan";
} else {
  echo "Error: " . $sql . "<br>" . $database->error;
}

$database->close();
```
