## Loop data memiliki range tertentu

Untuk loop data yang memiliki range tertentu dapat menggunakan fungsi for. Penggunaan fungsi for pada php, sebagai berikut:

```javascript
const namaBuah = [mangga, jeruk, apel, melon, salak]
let namaBuahLength = nama_buah.length

text = '<ul>'
for (let i = 0; i < namaBuahLength; i++) {
  text += '<li>' + namaBuah[i] + '</li>'
}
text += '</ul>'
```
