## Mengupdate data menggunakan PHP MySql

Cara mengupdate data ke dalam tabel profile menggunakan MySql, sebagai berikut:

```php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "ProfileTable";

// Membuat koneksi
$database = new mysqli($servername, $username, $password, $dbname);
// Mengecek koneksi
if ($database->connect_error) {
  die("Gagal terkoneksi: " . $database->connect_error);
}

$sql = "UPDATE Profile SET lastname='Silalahi' WHERE id=2";

if ($database->query($sql) === TRUE) {
  echo "Data baru berhasil terupdate";
} else {
  echo "Error: " . $database->error;
}

$database->close();
```
