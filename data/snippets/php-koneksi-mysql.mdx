## Menghubungkan data dari database menggunakan MySql

Sebelum kita dapat mengakses data dari database, kita perlu menghubungkan ke server dengan cara, sebagai berikut:

```php
$servername = "localhost";
$username = "username";
$password = "password";

// Membuat koneksi
$database = new mysqli($servername, $username, $password);

// Mengecek koneksi
if ($database->connect_error) {
  die("Gagal terkoneksi: " . $database->connect_error);
}
echo "Berhasil terkoneksi";
```
