## Fungsi Pushing

Array push berfungsi untuk memasukkan array baru ke dalam array. Penggunaan fungsi array push pada php, sebagai berikut:

```javascript
const namaBuah = [mangga, jeruk, apel, melon, salak]
namaBuah.push('kiwi') // Menambahkan "kiwi" ke namaBuah
```
