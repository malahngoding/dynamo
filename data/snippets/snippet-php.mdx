## Hypertext Preprocessor

#### Fungsi if dan else

Fungsi if dan else adalah suatu fungsi untuk menentukan suatu aksi dalam suatu kondisi. Cara menentukan fungsi if dan else pada php, sebagai berikut:

```php
if ($a > $b){
echo "a lebih besar dari b"
} else{
echo "a tidak lebih besar dari b"
};
```

#### Fungsi elseif

Fungsi elseif adalah suatu fungsi untuk menentukan suatu aksi dalam beberapa kondisi. Cara menentukan fungsi elseif pada php, sebagai berikut:

```php
if ($a > $b) {
    echo "a lebih besar dari b";
} elseif ($a == $b) {
    echo "a sama dengan b";
} else {
    echo "a lebih kecil dari b";
}
```

#### Fungsi Array Push

Array push berfungsi untuk memasukkan array baru ke dalam array. Penggunaan fungsi array push pada php, sebagai berikut:

```php
$names = array();
array_push($names, 'David', 'John');
```

#### Fungsi Array merge

Array merge berfungsi untuk menggabungkan dua array atau lebih. Penggunaan fungsi array merge pada php, sebagai berikut:

```php
$merged = array_merge($arr1, $arr2);
```

#### Fungsi count

Count berfungsi untuk menghitung jumlah array. Penggunaan fungsi count pada php, sebagai berikut:

```php
$nama_buah = [mangga, jeruk, apel, melon, salak];
count($nama_buah);
```

#### Loop data memiliki range tertentu

Untuk loop data yang memiliki range tertentu dapat menggunakan fungsi for. Penggunaan fungsi for pada php, sebagai berikut:

```php
for ($i = 0; $i <= 3; $i++) {
        echo "Loop data : " . $i;
    }
```

#### Loop data berupa array

Untuk loop data berupa array dapat menggunakan fungsi foreach. Penggunaan fungsi foreach pada php, sebagai berikut:

```php
$arr = array(1, 2, 3, 4);
foreach ($arr as &$value) {
    $value = $value * 2;
}
```

#### Menghubungkan data dari database menggunakan MySql

Sebelum kita dapat mengakses data dari database, kita perlu menghubungkan ke server dengan cara, sebagai berikut:

```php
$servername = "localhost";
$username = "username";
$password = "password";

// Membuat koneksi
$database = new mysqli($servername, $username, $password);

// Mengecek koneksi
if ($database->connect_error) {
  die("Gagal terkoneksi: " . $database->connect_error);
}
echo "Berhasil terkoneksi";
```

#### Memasukkan data menggunakan MySql

Cara memasukkan data ke dalam tabel profile menggunakan MySql, sebagai berikut:

```php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "ProfileTable";

// Membuat koneksi
$database = new mysqli($servername, $username, $password, $dbname);
// Mengecek koneksi
if ($database->connect_error) {
  die("Gagal terkoneksi: " . $database->connect_error);
}

$sql = "INSERT INTO Profile (firstname, lastname, email)
VALUES ('Galih', 'Indrasari', 'galihindrasari@gmail.com')";

if ($database->query($sql) === TRUE) {
  echo "Data baru berhasil tersimpan";
} else {
  echo "Error: " . $sql . "<br>" . $database->error;
}

$database->close();
```

#### Mengupdate data menggunakan MySql

Cara mengupdate data ke dalam tabel profile menggunakan MySql, sebagai berikut:

```php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "ProfileTable";

// Membuat koneksi
$database = new mysqli($servername, $username, $password, $dbname);
// Mengecek koneksi
if ($database->connect_error) {
  die("Gagal terkoneksi: " . $database->connect_error);
}

$sql = "UPDATE Profile SET lastname='Silalahi' WHERE id=2";

if ($database->query($sql) === TRUE) {
  echo "Data baru berhasil terupdate";
} else {
  echo "Error: " . $database->error;
}

$database->close();
```
